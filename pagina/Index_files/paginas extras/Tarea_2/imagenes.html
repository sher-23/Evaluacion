Sintaxis de imagenes HTML

El elemento IMG sirve para colocar imagenes en una pagina web


Las imagenes nose insertan en una pagina web sino que las imagenes estan siendo vinculadas atraves de un enlace


La etiqueta IMG esta vacia, solo contiene atributos y no tiene una etiqueta de cierre


La etiqueta IMG tiene dos atributos principales:


SRC: Especifica la ruta de la imagen


ALT: Especifica el texto alternativo de la imagen
